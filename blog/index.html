Xita is Public and Open Source

July 21, 2021

Cody Fagley

Xita Code Repository

For the last four and a half years, the Codeus team has worked on Xita as a closed source project. However, this was changed one month ago and Codeus has shifted its efforts to ensuring the Xita project is completed as quickly as possible.

If you are unfamiliar with the Xita project, it is an ambitious attempt to redesign foundational computing. It enforces "zero trust" policies on all OS, hardware, and firmware resources; this means that no one is able to directly access the backend data.

If you are interested in helping this effort, please get ahold of Cody Fagley through LinkedIn or via Email.
